レシピの作成方法

定形レシピ

event.shaped( //定形レシピの指定
Item.of('minecraft:stone', 8) //完成したもの
[
'AAA',
'ABA', //レシピの配置
'AAA'
],
{
A: 'minecraft:cobblestone',
B: 'minecraft:coal' //材料
}

不定形レシピ

event.shapeless(
Item.of('minecraft:grass', 3), //完成したもの
[
'minecraft:bone_meal',
'minecraft:dirt' // 材料
]
)

鍛冶台のレシピ (ver.1.20以上)

event.smithing(//鍛冶台のレシピ
'minecraft:netherite_ingot', // 完成品
'minecraft:netherite_upgrade_smithing_template', // 鍛冶型の部分
'minecraft:iron_ingot', // アップグレード
'minecraft:black_dye' // 完成前の素材
)

鍛冶台のレシピ (ver.1.19.2以下)

event.smithing(//鍛冶台のレシピ
'minecraft:netherite_ingot', // 完成品
'minecraft:iron_ingot', // アップグレード
'minecraft:black_dye' // 完成前の素材
)
