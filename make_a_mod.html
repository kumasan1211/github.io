レシピの作成方法

定形レシピ

event.shaped( //定形レシピの指定
"3x minecraft:cobblestone" //完成したもの
[
'A B',
' C ', //レシピの配置
'B A'
],
{
A: 'minecraft:andesite',
B: 'minecraft:diorite', //材料
C: 'minecraft:granite'
}

不定形レシピ

event.shapeless(
Item.of('minecraft:grass', 3), //完成したもの
[
'minecraft:bone_meal',
'minecraft:dirt' // 材料
]
)
